gao@DESKTOP-O0IOHNI MINGW64 /e/project/demo/weijinsuo (master)

##将当前目录下所有文件添加到仓库
gao@DESKTOP-O0IOHNI MINGW64 /e/project/demo/weijinsuo (master)
$ git add –all

$ git status

Initial commit

(use “git rm –cached …” to unstage)

##提交到本地仓库中
gao@DESKTOP-O0IOHNI MINGW64 /e/project/demo/weijinsuo (master)

##添加仓库上传的远程地址
gao@DESKTOP-O0IOHNI MINGW64 /e/project/demo/weijinsuo (master)
$ git remote add origin https://github.com/FreedomPanda/weijinsuo.git
